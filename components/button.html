# 按钮

📌 Attributes

组件将被渲染为 button标签，因此你可以使用所有的 button原生属性。

参数 说明 类型 可选值 默认值
icon icon 图标名 String —— ——
position icon 位置 String left , right left
isLoading 带 loading 效果的按钮，当按钮被点击时会出现 loading 效果，再次点击 loading 效果消失 Boolean —— false
